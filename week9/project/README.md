# Student Registration System
#### Video Link: https://youtu.be/6pjcjX1xnUc


#### Description:
This is a Python program designed for managing student registration.
It allows staff members to verify existing students and register new ones.

1. Run the `main.py` file:
2. Enter the student number when prompted. The Number must start with '10' and end with any number not 0
3. If the student is registered, their name and surname will be displayed.
4. If the student is not registered, The console will ask you if you want register this new student, If you input 'Y' It will ask you to verify you are staff you will have the option to register them if you are a staff member. It will prompt for staff to enter the student number then another input for name and another input for surname
5. But if you Input 'N' instead of 'Y' the program will exit with sys.exit
6. When new student was registered the it will print the 'New Student Registered'

### Prerequisites
- Python 3.x installed on your machine


## Features

- Verify Students: Staff members can verify existing student IDs.
- Register Students: Staff members can register new students.
- Simple Interface: Command-line interface for ease of use.
